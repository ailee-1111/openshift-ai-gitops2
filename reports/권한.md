## 최소 권한 원칙

- poc-user(DS)는 모델 등록과 서빙만 수행. 배포는 poc-operator(MGR)가 담당
- poc-admin이 5개 시나리오(S5, S6, S8, S11 일부)에서 cluster-admin 사용. PoC 특성상 허용하되, 프로덕션 전환 시 권한 축소 권장
- 시연 시 `oc get secret -o yaml` 사용 금지 (비밀번호 평문 노출 방지)
- 데모 후 MaaS API 키 즉시 폐기
