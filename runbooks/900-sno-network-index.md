# 900 — SNO 접근 간헐적 끊김 트러블슈팅 (30스프린트 × 5방법)

## 아키텍처

```
┌──────────┐   VPN    ┌───────────────────┐        ┌─────────────┐
│  노트북   │ ───────► │  Bastion 서버      │ ──────► │  SNO 클러스터 │
│          │  tunnel  │  - DNS (named/dnsmq)│  direct │  (API:6443)  │
│ hosts 파일│         │  - HAProxy (6443,   │        │  (*.apps)    │
│ → bastion│         │    443, 80)         │        │              │
└──────────┘         └───────────────────┘        └─────────────┘
```

**증상**: VPN 연결 후 `oc` 명령 또는 콘솔 접근 시 간헐적 지연·타임아웃·끊김
